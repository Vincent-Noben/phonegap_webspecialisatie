Install NodeJS

  • Browse to nodejs.org
  • Inform yourself about NodeJS
  • Click the install button
  • Download and run the installation file
  • Click next/accept/continue/...
    and wait for nodejs to finish installation
  • Open your terminal/commandprompt
  • Type the following command to test Node is up and running 
    								$ node -v
  • Type the following command to test Node Package Manager is up and running 
    								$ npm -v
